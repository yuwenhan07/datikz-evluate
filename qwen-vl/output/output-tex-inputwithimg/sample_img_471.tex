        \draw[black, thick] (A) .. controls (B) and (C) .. (D);

The line segment $s(t)$ connecting $x_0$ and $x_1$ is presented by the red line. The tilted interval box is centered at $s(t)$ at each $t\in [t_0,t_1]$ with the same radius.}
