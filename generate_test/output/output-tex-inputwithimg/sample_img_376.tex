\section*{Example Construction from Theorem~\ref{thm:const}}

The following diagram illustrates an example of the construction described in Theorem~\ref{thm:const}. It shows a tournament on the left and the corresponding vertex coloring on the right.

\begin{tikzpicture}[->,>=Stealth',shorten >=1pt,auto,node distance=2cm,
  thick,main node/.style={circle,fill=#1!50,draw,font=\sffamily\Large\bfseries}]
  \node[main node=yellow] (1) {$1$};
  \node[main node=blue] (2) [below right of=1] {$2$};
  \node[main node=red] (3) [below left of=2] {$3$};
  \node[main node=black] (4) [above right of=1] {$4$};

  \path[every node/.style={font=\sffamily\small}]
    (1) edge node [left] {} (2)
        edge node [] {} (3)
    (2) edge node [] {} (4)
        edge node [] {} (1)
    (3) edge node [] {} (4)
        edge node [] {} (1)
    (4) edge node [] {} (3)
        edge node [] {} (2);
\end{tikzpicture}
