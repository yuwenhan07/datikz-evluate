To generate the basis functions for the space \( S^3_0 \) with the given knot vector \(\{0, 0, h, h, 7h, 7h, 1, 1\}\), we can use B-spline basis functions. The B-spline basis functions are defined recursively using the Cox-de Boor recursion formula.

Here is the TikZ LaTeX code to visualize these basis functions:

\pgfplotsset{compat=1.16}
